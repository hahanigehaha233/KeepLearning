### 如何找出链表中环的起始节点
```
设a为链表起始到环起始的长度，x为环起始到快慢指针相遇的长度，y为快慢指针相遇到环
起始的长度，r为环的长度，S为慢指针所走的长度，则
S = a + x
2S = a + x + nr
所以： S = nr
S = nr = (n - 1) r + r = (n - 1) r + x + y
所以: a + x = (n - 1) r + x + y
-> a = (n - 1) r + y  = nr - x 则a的长度等于当前fast指针位置再走(n-1)r+y步
```
